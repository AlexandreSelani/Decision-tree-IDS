Código utilizado para exemplificar um IDS (Intrusion Detection System) utilizando o algoritmo de árvores de decisão.

Como executar:
  1. Instalar as bibliotecas em requirements.txt
  2. Baixar o dataset do link: https://www.kaggle.com/datasets/subhajournal/iotintrusion
  3. Colocar o dataset na pasta do repositório
  4. Executar "decision_tree.py"
